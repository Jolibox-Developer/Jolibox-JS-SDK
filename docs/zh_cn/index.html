广告 SDK

Jolibox SDK 模块使您可以通过广告赚钱。目前版本我们简单地封装了 Google Ad Sense 和 Ads for Game。

大多数功能严格遵循广告投放 API 设计。您可以阅读 Google 的官方文档 这里

当前版本仍处于预发布阶段，因此 API 可能会在未来发生变化。

  • 安装 SDK

    npm install @jolibox/ads
