此广告位的状态，可以是以下值之一：

breakStatus 原因
'notReady' 广告位 API 尚未初始化
'timeout' 由于广告位 API 响应时间过长，广告位超时
'invalid' 广告位无效并被忽略——例如每次页面加载只应有一个前贴广告，后续的前贴广告会因该状态失败
'error' 回调中出现了 JavaScript 错误
'noAdPreloaded' 尚未预加载广告，因此跳过此广告位
'frequencyCapped' 未显示广告，因为此广告位的频率限制已应用
'ignored' 用户在到达下一个广告位之前没有点击奖励提示，即 showAdFn() 没有在下一个 adBreak() 之前调用
'other' 由于其他原因未显示广告。（例如，广告仍在获取中，或者由于屏幕调整大小/旋转，之前缓存的广告已被处理掉。）
'dismissed' 用户在观看完广告之前关闭了奖励广告
'viewed' 用户已观看广告
