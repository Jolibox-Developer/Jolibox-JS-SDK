afterAd?: (() => void)

(可选) 广告结束后（无论出于何种原因）调用。对于奖励广告，在 adDismissed 或 adViewed 之后调用，具体取决于玩家的操作。此函数应用于恢复游戏流程。例如，用于取消静音并开始下一关。

可选beforeAd

beforeAd?: (() => void)

(可选) 在广告显示之前调用。游戏应暂停并静音。这些操作必须同步完成。广告将在此回调结束后立即显示。

可选name

name?: string

(可选) 此广告位在游戏中的名称。它是一个内部标识符，不会显示给玩家。未来的版本中，可能会使用此标识符来启用额外的报告和优化功能。

我们建议您为所有广告位命名。
